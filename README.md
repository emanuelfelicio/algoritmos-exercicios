# Algoritmos e Resolução de Problemas

Este repositório reúne estudos sobre algoritmos e estruturas de dados, com foco no aprimoramento da lógica e no domínio de conceitos fundamentais da computação.

O objetivo é exercitar a análise de problemas e a aplicação de soluções eficientes, utilizando desafios baseados em roteiros como o NeetCode e LeetCode.

## Foco de Estudo

A prática aqui se concentra em três pilares principais:

- **Eficiência:** Escolha de estruturas de dados e algoritmos com foco no uso otimizado de tempo e memória.
- **Decomposição de Problemas:** Técnica de dividir desafios complexos em componentes menores e mais gerenciáveis.
- **Fundamentos:** Prática constante dos conceitos base que sustentam o desenvolvimento de software.

## Organização

Para facilitar o fluxo de estudos, utilizo um utilitário simples para geração do padrão inicial de novos arquivos:

```bash
go run script/main.go "Nome do Exercicio"
```
